== Description ==

그누페이는 워드프레스 쇼핑몰 플러그인 Woocommerce 와 한국 PG ( NHN KCP ) 결제시스템과 연동하여 결제할수 있는 플러그인입니다.

우커머스는 워드프레스에서 사용자층이 매우 두터운 쇼핑몰 플러그인 입니다.
국내에서도 우커머스에 대한 관심이 높아 점차 사용자가 많아지고 있는 추세입니다.
이 추세에 맞춰 저희 SIR에서는 우커머스에서 결제를 할수 있는 플러그인을 개발하게 되었습니다.
 
우커머스에서만 작동되는 이 한국형 결제 플러그인의 이름이 그누페이 입니다.

포럼 : http://sir.kr/main/gnupay/  

자주하시는질문 : http://sir.kr/faq?s_tag=%EA%B7%B8%EB%88%84%ED%8E%98%EC%9D%B4

결제 데모 사이트 : http://demo.sir.kr/gnupay/?post_type=product

== Installation ==

이 섹션은 그누커머스를 설치하여 사용하는 방법에 대해 설명하고 있습니다.

1. 이 플러그인은 설치하기 이전에, 우커머스 플러그인이 활성화 되어 있어야 합니다.
2. `/wp-content/plugins/` 디렉토리에 `gnupay` 폴더를 만들고 업로드 합니다.
3. 관리자 `플러그인` 메뉴에서 GNUPAY를 활성화 합니다.

This section describes how to install the plugin and get it working.

1. It requires Woocommerce plugin.
2. Upload `gnupay` folder in your `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= 그누페이 - 전자결제 =

그누페이는 영카트5에서 사용하는 결제대행사를 모두 사용할수 있도록 프로그램 되어 있습니다.
현재 NHN KCP 만 가능합니다.
물론, 영카트5와 수수료 및 서비스 내용은 동일합니다.
 
결제대행사는 부가서비스 메뉴에서 원하는 회사를 선택하여 신청하시면 됩니다.
http://sir.co.kr/main/service/ 
 
신청후 상점아이디를 받으셨다면 그누페이 매뉴얼의 'NHN KCP 카드결제 설정' 페이지를 참고하여 설정하시면 됩니다.
http://sir.kr/manual/gnupay/209
